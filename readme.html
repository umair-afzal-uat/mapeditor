# WordPress Plugin: Mapeditor This WordPress plugin allows users to create custom maps, redimate maps, connect with Strava, and generate files via GPX for export in multiple formats. Additionally, it integrates with WooCommerce for a seamless user experience. ## Features - **Custom Maps Creation**: Create custom maps with various layers, markers, and routes tailored to your needs. - **Redimate Maps**: Generate redimate-style maps with enhanced design and customization options. - **Strava Integration**: Connect your Strava account to the plugin to create maps based on your Strava activities. - **GPX File Export**: Import GPX files, edit them, and export them in various formats including GPX, KML, or others. - **WooCommerce User Integration**: Seamlessly integrate with WooCommerce to allow users to access maps or features related to products or services. ## Installation 1. Download the plugin zip file. 2. In your WordPress dashboard, go to **Plugins > Add New**. 3. Click **Upload Plugin** and select the downloaded zip file. 4. Click **Install Now** and then **Activate**. ## Usage Once the plugin is activated, you can begin creating custom maps by navigating to the plugin settings in the WordPress dashboard. Here's how you can use the key features: - **Custom Map Creation**: Go to the **Custom Maps** section and use the map builder to add markers, shapes, and routes. - **Redimate Maps**: Access the **Redimate Maps** section for specialized map designs and customization. - **Strava Integration**: Connect your Strava account in the plugin settings to import and create maps based on your Strava data. - **GPX File Handling**: Upload a GPX file via the **Import GPX** section, make necessary adjustments, and then export in your preferred format. ## WooCommerce Integration - The plugin will automatically link with WooCommerce to enhance user interaction. It enables specific map features tied to WooCommerce products or services, giving your customers access to customized mapping features tied to their purchases. ## Requirements - WordPress 6.0 or higher - WooCommerce (if using WooCommerce integration) - PHP 7.4 or higher - An active Strava account (for Strava map creation) ## Support If you encounter any issues or need further assistance, please feel free to open an issue on our GitHub page or contact support at [support@example.com]. ## License This plugin is released under the [MIT License](LICENSE).
